# Post-COVID-Health-Effects
End-to-end data analysis project using Python to explore recovery patterns and factors influencing Long COVID risk. Includes EDA, feature engineering, and interpretable machine learning with Logistic Regression, emphasizing insight generation and methodological correctness.

Objectives
- Analyze recovery time across different COVID severity levels  
- Study the relationship between symptoms and Long COVID risk  
- Evaluate the impact of hospitalization and physical activity  
- Understand mental health outcomes after COVID  
- Present insights through an interactive dashboard  

## Dataset
- **Records:** 500 patients  
- **Format:** CSV  
- **Domain:** Healthcare  
- **Key Features:**
  - Demographics (Age, Gender)
  - COVID experience (Severity, Hospitalization)
  - Recovery metrics (Days to Recovery)
  - Symptoms (Fatigue, Brain Fog, Breathing Issues, Loss of Taste/Smell)
  - Lifestyle factors (Physical Activity)
  - Mental health impact
  - Long COVID risk category


## Tech Stack
- **Programming:** Python  
- **Data Analysis:** Pandas, NumPy  
- **Visualization:** Matplotlib, Seaborn  
- **Dashboard:** Streamlit  
- **Environment:** Linux
